## 페이지 5: 입찰 작업 화면 (BidWorkspace) ⭐ 핵심

**경로**: `/bids/{id}/workspace`
**목적**: 장표 조합, 편집, 최종 PDF 생성

```
┌──────────────────────────────────────────────────────────┐
│  OO부 정보시스템 구축                                      │
│  발주처: OO시청 │ 공고: 2026-001 │ 마감: 3/15 │ [설정⚙]  │
├─────────────────────┬────────────────────────────────────┤
│                     │                                    │
│  📄 장표 레이어      │  선택됨: 참여인력 현황표              │
│                     │                                    │
│  ≡ 1. 참여인력 현황표│  [편집] [AI수정] [미리보기]          │
│     [HTML] ✅       │                                    │
│  ≡ 2. 김OO 경력증명 │  ┌──────────────────────────────┐  │
│     [HTML] ✅       │  │                              │  │
│  ≡ 3. 이OO 경력증명 │  │                              │  │
│     [HTML] ✅       │  │    (A4 미리보기)              │  │
│  ≡ 4. 사업자등록증   │  │                              │  │
│     [PDF] 1p        │  │    표 형태의 인력현황표가       │  │
│  ≡ 5. 재무제표      │  │    실제 양식과 동일하게 보임     │  │
│     [PDF] 3p        │  │                              │  │
│  ≡ 6. 실적증명서    │  │                              │  │
│     [PDF] 1p        │  │                              │  │
│                     │  └──────────────────────────────┘  │
│  ─────────────      │                                    │
│  [+ 새 HTML 장표]   │  👤 배정 인력                       │
│  [+ PDF 업로드]     │  ┌──────────────────────────────┐  │
│  [+ AI 양식 생성]   │  │ 김OO (PM) - 이력 3건 [편집]   │  │
│  [+ 인력 장표 생성] │  │ 이OO (PL) - 이력 2건 [편집]   │  │
│                     │  └──────────────────────────────┘  │
│  ═════════════      │  [+ 인력 배정]                      │
│  📊 총 6개 장표     │                                    │
│  📄 총 8페이지      │                                    │
│                     │                                    │
│  [📥 최종 PDF 생성] │                                    │
└─────────────────────┴────────────────────────────────────┘
```

**좌측 패널 (장표 레이어)**:
- PageLayerPanel 컴포넌트
- 각 장표가 카드로 표시 (≡ 핸들로 드래그)
- [HTML] 또는 [PDF] 배지로 타입 구분
- ✅ 는 PDF 생성 완료 표시
- 페이지 수 표시
- 하단: 추가 버튼 4종
  - 새 HTML 장표: 빈 장표 추가
  - PDF 업로드: 파일 선택 다이얼로그
  - AI 양식 생성: PDF 업로드 → Gemini로 HTML 생성
  - 인력 장표 생성: 인력 선택 → 자동 HTML 생성
- 최하단: 요약 정보 + 최종 PDF 생성 버튼

**우측 패널 (미리보기/편집)**:
- 선택된 장표의 미리보기 (A4 비율)
- HTML 장표: [편집] 클릭 → PageEditor로 이동
- HTML 장표: [AI수정] 클릭 → AI 수정 모달
- PDF 장표: PDF 뷰어로 표시
- 하단: 배정 인력 섹션

**드래그앤드롭 동작**:
- 장표 카드의 ≡ 핸들을 잡고 위/아래로 드래그
- 드롭 시 sort_order 자동 업데이트 (API 호출)
- 시각적 피드백 (드래그 중 반투명, 드롭 위치 표시)

## 페이지 6: 장표 편집 (PageEditor)

**경로**: `/bids/{bid_id}/pages/{page_id}/edit`
**목적**: HTML 장표의 코드 편집, AI 수정, 실시간 미리보기

```
┌──────────────────────────────────────────────────────────┐
│  참여인력 현황표 편집                    [저장] [닫기]     │
├──────────┬───────────────────┬───────────────────────────┤
│          │                   │                           │
│  💬 AI   │  </> 코드 편집     │  👁 미리보기              │
│  채팅    │                   │                           │
│          │  [HTML] [CSS] 탭  │  ┌─────────────────────┐  │
│ ─────── │                   │  │                     │  │
│          │  <html>           │  │  A4 크기로 렌더링    │  │
│ 사용자:  │  <head>           │  │                     │  │
│ "3번째   │    <style>        │  │  코드 변경 시        │  │
│  열 너비 │    ...            │  │  실시간 반영         │  │
│  넓혀줘" │    </style>       │  │                     │  │
│          │  </head>          │  │  ┌──┬──┬──┬──┐     │  │
│ AI:      │  <body>           │  │  │순│성│직│자│     │  │
│ "3번째   │    <table>        │  │  │번│명│급│격│     │  │
│ 열 너비를│      <tr>         │  │  ├──┼──┼──┼──┤     │  │
│ 30%로   │        <th>순번    │  │  │1 │김│부│기│     │  │
│ 변경함"  │        <th>성명    │  │  │  │OO│장│술│     │  │
│          │        ...        │  │  │  │  │  │사│     │  │
│          │                   │  │  └──┴──┴──┴──┘     │  │
│          │                   │  │                     │  │
│          │                   │  └─────────────────────┘  │
│ [입력..] │                   │                           │
└──────────┴───────────────────┴───────────────────────────┘
```

**3분할 레이아웃**:
- 좌: AI 채팅 패널 (자연어로 수정 요청)
- 중: Monaco 코드 편집기 (HTML/CSS 탭 전환)
- 우: 실시간 미리보기 (iframe, A4 비율)

**AI 채팅 패널**:
- 사용자가 자연어로 입력 → API 호출 → 수정된 코드 반영
- 대화 이력 유지 (세션 내)
- 수정 전/후 비교 가능

**코드 편집기 (Monaco)**:
- HTML 탭 / CSS 탭 전환
- 구문 강조 (Syntax Highlighting)
- 자동 완성
- 코드 변경 시 미리보기 실시간 반영 (debounce 300ms)

**미리보기**:
- iframe으로 HTML/CSS 렌더링
- A4 비율 고정 (210mm x 297mm 비율)
- 확대/축소 가능
